                <h2 id="favorite-enemy">🏹 Избранный враг (Favored Enemy)</h2>
                <p>+1d4 урона по Избранным врагам</p>

                <p>Ты специализируешься на охоте за определёнными видами существ. Выбери один тип избранного врага из следующего списка:</p>
